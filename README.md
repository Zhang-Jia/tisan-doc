# acib-doc  
*** 
  
[acib](http://ide.acloudiot.com/) 是的可产品化的物联网开发板，有硬件（acib开发板，当前核心模块基于ESP8266，一款SOC的WIFI芯片）以及软件（固件、云端编辑WebIDE、手机应用程序ACloudIot、微信应用、平台服务等）。

| -------- |:--------------|:-----------|:-------------------------|
